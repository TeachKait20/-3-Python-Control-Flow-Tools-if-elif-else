## –í–µ—Ç–≤–ª–µ–Ω–∏–µ –≤ Python

–í–µ—Ç–≤–ª–µ–Ω–∏–µ –≤ Python ‚Äî —ç—Ç–æ –æ—Å–Ω–æ–≤–Ω–∞—è –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –ø–æ—Ç–æ–∫–æ–º –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –ø—Ä–æ–≥—Ä–∞–º–º—ã, –ø–æ–∑–≤–æ–ª—è—é—â–∞—è –≤—ã–ø–æ–ª–Ω—è—Ç—å —Ä–∞–∑–ª–∏—á–Ω—ã–µ –±–ª–æ–∫–∏ –∫–æ–¥–∞ –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —É—Å–ª–æ–≤–∏–π. –û—Å–Ω–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –≤–µ—Ç–≤–ª–µ–Ω–∏—è –≤ Python –≤–∫–ª—é—á–∞—é—Ç `if`, `elif`, `else`, –∞ —Ç–∞–∫–∂–µ `match-case` (–∞–Ω–∞–ª–æ–≥ `switch-case`, –¥–æ—Å—Ç—É–ø–µ–Ω —Å Python 3.10). 

### –û–ø–µ—Ä–∞—Ç–æ—Ä `if`

–û–ø–µ—Ä–∞—Ç–æ—Ä `if` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –±–ª–æ–∫–∞ –∫–æ–¥–∞, –µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ. –ï—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –ª–æ–∂–Ω–æ, —ç—Ç–æ—Ç –±–ª–æ–∫ –ø—Ä–æ–ø—É—Å–∫–∞–µ—Ç—Å—è.

```Python
x = 10
if x > 0:
    print("x –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ")
```

### –û–ø–µ—Ä–∞—Ç–æ—Ä `else`

`else` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤–º–µ—Å—Ç–µ —Å `if` –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç –∫–æ–¥, –µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ `if` –ª–æ–∂–Ω–æ.

```Python
x = -5
if x > 0:
    print("x –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ")
else:
    print("x –Ω–µ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ")
```

### –û–ø–µ—Ä–∞—Ç–æ—Ä `elif`

`elif` –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —É—Å–ª–æ–≤–∏–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ. –í—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ –ø–µ—Ä–≤—ã–π –∏—Å—Ç–∏–Ω–Ω—ã–π –±–ª–æ–∫.

```Python
x = 0
if x > 0:
    print("x –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ")
elif x < 0:
    print("x –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ")
else:
    print("x —Ä–∞–≤–Ω–æ –Ω—É–ª—é")
```

## –¢–∞–±—É–ª—è—Ü–∏—è –≤ Python

–í–æ –≤—Å–µ—Ö –ø—Ä–∏–º–µ—Ä–∞—Ö –≤—ã—à–µ, –≤—ã –≤–∏–¥–µ–ª–∏ –æ—Ç—Å—Ç—É–ø—ã (4 –ø—Ä–æ–±–µ–ª–∞ –∏–ª–∏ 1 tab)

–¢–∞–±—É–ª—è—Ü–∏—è (–æ—Ç—Å—Ç—É–ø—ã) –≤ Python ‚Äî —ç—Ç–æ –∫–ª—é—á–µ–≤–æ–π —ç–ª–µ–º–µ–Ω—Ç —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–∞, –∫–æ—Ç–æ—Ä—ã–π –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –∫–æ–¥–∞ –∏ –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏—è –≤–ª–æ–∂–µ–Ω–Ω–æ—Å—Ç–∏ –±–ª–æ–∫–æ–≤.

```
        –í–ª–æ–∂–µ–Ω–Ω–æ–µ –¥–µ–π—Å—Ç–≤–∏–µ –∏–ª–∏ —É—Å–ª–æ–≤–∏–µ (–¥–≤–∞ —É—Ä–æ–≤–Ω—è –æ—Ç—Å—Ç—É–ø–∞)
```

**–ü—Ä–∏–º–µ—Ä –Ω–∞ –æ—Å–Ω–æ–≤–µ —Å—Ü–µ–Ω–∞—Ä–∏—è —Å –∂–∏–∑–Ω–µ–Ω–Ω–æ–π —Å–∏—Ç—É–∞—Ü–∏–µ–π:**

```Python
    if x % 2 == 0:                # –í–ª–æ–∂–µ–Ω–Ω–æ–µ —É—Å–ª–æ–≤–∏–µ, –Ω–æ–≤—ã–π —É—Ä–æ–≤–µ–Ω—å —Ç–∞–±—É–ª—è—Ü–∏–∏

    else:                         # –°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –≤–ª–æ–∂–µ–Ω–Ω–æ–º—É if
        print("–ß–∏—Å–ª–æ –Ω–µ—á–µ—Ç–Ω–æ–µ")   # –í—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ x –Ω–µ—á–µ—Ç–Ω–æ–µ
```

## –í–ª–æ–∂–µ–Ω–Ω—ã–µ —É—Å–ª–æ–≤–∏—è

–£—Å–ª–æ–≤–∏—è –º–æ–∂–Ω–æ –≤–∫–ª–∞–¥—ã–≤–∞—Ç—å –¥—Ä—É–≥ –≤ –¥—Ä—É–≥–∞ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è —Å–ª–æ–∂–Ω—ã—Ö –ª–æ–≥–∏–∫.
